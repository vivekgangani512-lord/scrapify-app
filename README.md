# Scrapify App

A multi-user Android scrap management application built using Java and XML.  
The app connects home owners with scrap dealers through a bidding workflow system.

## Features
- User & Scrap Dealer Authentication
- Scrap Listing & Category Management
- Deal Tracking System
- Bidding Workflow
- Dashboard UI for Dealers & Users
- Responsive RecyclerView-based Layouts
- SQLite Local Database Integration

## Tech Stack
- Java
- XML
- Android Studio
- SQLite
- RecyclerView
- Material UI
